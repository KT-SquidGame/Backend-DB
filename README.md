## Tutorial
해당 코드는 AI 오징어게임의 DB서버 코드입니다.
핵심 기능은 Client 에서 필요한 데이터 호출, 데이터 저장 요청이 올경우 Rest API 통신으로 실행이 됩니다.

<b> 실행 방법 </b>
1. 코드 다운
2. IDE에서 MainDbApplication 실행 (Inteliz IDEA 권고)
